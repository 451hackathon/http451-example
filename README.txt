Example configuration to test serving HTTP 451
==============================================

A handful of simple images and html pages, along with an nginx
configuration to try to make it appear blocked.

This should be installed someplace and used to test detection.

The images come from:

 * image0.png: [gtkperf](https://packages.debian.org/gtkperf): `/usr/share/pixmaps/duck.png`
 * image2.png: [openscad](https://packages.debian.org/openscad): `/usr/share/pixmaps/openscad.png`
 * image1.png: [calibre](https://packages.debian.org/calibre): `/usr/share/pixmaps/calibre-gui.png`
 * blocked-body.png: [oxygen-icon-theme](https://packages.debian.org/oxygen-icon-theme): `/usr/share/icons/oxygen/base/22x22/emotes/opinion-no.png`
